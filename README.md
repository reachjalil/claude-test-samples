# Sample Claude Exams

Standalone Astro site for independent sample Claude exam practice.

## What Is Included

- Timed, untimed, targeted, and flashcard practice modes.
- Static question-bank pages with answer explanations.
- Study path pages mapped to domains, scenarios, and task statements.
- Public disclaimer, privacy policy, and terms pages.
- Local-browser progress storage with no required account.

## Local Development

```bash
pnpm install
pnpm dev
```

## Validation

```bash
pnpm check
pnpm build
```

## Deploy

This repository is configured for Cloudflare Workers Static Assets.

```bash
pnpm run deploy
```
